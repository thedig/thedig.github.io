BearTask - Single Page application with Backbone.js

BearTask is a task management system using list and card management. It leverages jQuery and Backbone.js for a single page application user experience. The application utilizes BDD to ensure that the application can expand with new features without disrupting the app's expected behavior. The server side code-base is in Rails 3 and Postgresql.
